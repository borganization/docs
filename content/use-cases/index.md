# Use Cases

Concrete things people do with Borg. Each page lists setup, gateway binding, and example prompts.

- [personal-assistant](personal-assistant) — calendar, email, todos, via iMessage or Telegram
- [developer-copilot](developer-copilot) — coding agent in a terminal
- [ops-bot](ops-bot) — Slack bot that runs scripts, reads logs, pages on-call
- [cross-device-notes](cross-device-notes) — capture from phone, read on laptop
- [home-automation](home-automation) — shell out to HomeAssistant or shortcuts
- [research-assistant](research-assistant) — web browse, summarize, remember
