# Slash Commands

Slash commands drive Borg: settings, models, status, memory, plugins. Type `/` inside any session to trigger.

## Start here

| Command | Use |
|---|---|
| `/help` | List every command |
| `/settings` | Open the settings popup |
| `/model` | Switch LLM provider and model |
| `/plugins` | Add, remove, manage plugins and channels |
| `/projects` | Browse and switch projects |
| `/memory` | Inspect long-term memory context |
| `/status` | Vitals: stability, focus, sync, growth, happiness |
| `/clear` | Clear the session |
| `/quit` | Exit |
