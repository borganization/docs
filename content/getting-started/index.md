# Getting Started

Three pages. Read top to bottom.

- [install](install) — one-line installer, manual download, build from source
- [onboarding](onboarding) — picking a provider, personality, where your data lives
- [providers](providers) — supported LLMs and how to pick one
- [slash-commands](slash-commands) — everyday commands you will use in session

## TL;DR

```sh
curl -fsSL https://raw.githubusercontent.com/borganization/borg/main/scripts/install.sh | bash
borg
```

Paste an API key when prompted. Start chatting.
